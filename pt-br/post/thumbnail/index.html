Thumbnail no post

Aug 6, 2023 Um minuto en pt-br

Agora você pode adicionar thumbnails no seu post!

Para adicionar você só precisa colocar a variável thumbnail no front-matter, ela pode ter os seguintes parâmetros:

Parâmetro Descrição Default
src Source da imagem, pode ser qualquer URL ou path relativo
alt O alt da imagem, pode ser qualquer string thumbnail
object_postion Valores que são passados para a propriedade object-position da imagem 50% 50%
height A altura do container da imagem 1 250px

Examplo

O thumbnail desse post foi gerado por esse trecho no front-matter:

thumbnail:
    src: 'images/thumbnail.jpg'
    alt: 'Post thumbnail'
    object_position: '50% 100%'
    height: 250px

Créditos

Imi – o personagem desse thumbnail – foi fornecido graciosamente por TD Vadilho, você pode checar o trabalho dele no site e YouTube!

  1. A altura padrão é 250px. Ela está definida na variável --thumbnail-size de CSS, você pode facilmente sobrescrever globalmente com seu CSS customizado. ↩︎
